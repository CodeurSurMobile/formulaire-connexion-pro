# Formulaire de connexion – Version Pro

Formulaire avec BEM, variables CSS et classes utilitaires.

Caractéristiques
- **BEM** : code propre et maintenable
- **Variables CSS** : facile à personnaliser
- **Utilitaires** : classes réutilisables (flexbox, animations)
- **Responsive** : s'adapte à toutes les tailles d'écran
- **Accessible** : labels, contrastes, autocomplete

Technologies
- HTML5
- CSS3 (BEM, variables, flexbox)

Fichiers
- `index.html` : structure du formulaire
- `styles.css` : styles principaux
- `variables.css` : variables CSS
- `utilitaires.css` : classes utilitaires
